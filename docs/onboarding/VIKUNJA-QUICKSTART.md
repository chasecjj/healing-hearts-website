## Logging In

1. Open your browser and go to: **http://10.0.0.100:3456**
2. Enter your username and password — **ask Chase** if you don't have these yet
3. You'll see your dashboard with any projects you have access to

> **Important:** Vikunja runs on PHEDRIS (the home server). You need to be on the same network to access it. If you're connecting remotely, ask Chase about VPN access.

> **What if the page won't load?** Make sure you're connected to the home network. Try refreshing the page. If it still won't load, PHEDRIS might be restarting — wait a minute and try again.

## Viewing Tasks

### List View (Default)

When you open a project, you'll see tasks listed with:
- **Title** — what needs to be done
- **Priority** — how urgent it is (see below)
- **Due date** — when it should be finished
- **Assignee** — who's responsible (look for your name)

### Kanban View

Kanban shows tasks as cards in columns (like sticky notes on a board). To switch views:
1. Look for a **view toggle** near the top of the task list
2. Click **Kanban** or the board icon
3. Tasks are organized by status (To Do, In Progress, Done)

You can drag cards between columns to update their status.

### Filtering to Your Tasks

To see only tasks assigned to you:
1. Look for a **filter** or **search** option near the top of the task list
2. Filter by **Assignee** and select your name
3. This shows only the tasks you're responsible for

## Working with Tasks

### Marking a Task Complete

When you finish a task:
1. Find the task in the list
2. Click the **checkbox** next to the task title
3. The task moves to the "Done" state

That's it! Chase and the rest of the team can see that it's finished.

### Adding a Comment

If you have a question about a task, or want to note your progress:
1. Click on the task to open it
2. Scroll down to the **Comments** section
3. Type your message and press **Send** or **Enter**

Use comments for things like:
- "I have a question about this — what do you mean by [X]?"
- "Started working on this, should be done by tomorrow"
- "Finished the first draft, ready for review"

### Creating a New Task

If you notice something that needs to be done:
1. Open the relevant project (e.g., Healing Hearts)
2. Click the **"+" button** or **"New Task"** option
3. Give it a clear title (e.g., "Update FAQ with question about session length")
4. Add a description if helpful
5. Set a priority if you know how urgent it is
6. Save

> **Tip:** When in doubt about which project a task belongs in, put it in Healing Hearts or ask Chase.

## Understanding Task Details

### Priority Levels

| Priority | What It Means | Visual |
|----------|--------------|--------|
| **Urgent** | Needs attention today or ASAP | Red or exclamation mark |
| **High** | Should be done this week | Orange |
| **Medium** | Normal priority, do when you can | Yellow or default |
| **Low** | Nice to have, no rush | Blue or gray |
| (No priority) | Not yet triaged — Chase will set it | No indicator |

### Due Dates

- Tasks with due dates show the date next to the title
- **Overdue tasks** are highlighted (usually in red)
- If you can't meet a due date, add a comment explaining why and what your new estimate is — that's totally fine

### Labels

Tasks may have labels (colored tags) to categorize them, like:
- **Content** — writing or editing text
- **Design** — visual/layout changes
- **Bug** — something is broken and needs fixing
- **Review** — needs someone to look it over

## Daily Workflow

Here's how to use Vikunja as part of your daily work:

### When You Start Working

1. **Log in** to Vikunja at http://10.0.0.100:3456
2. **Check your tasks** — filter to tasks assigned to you
3. **Pick one** — start with the highest priority or the closest due date
4. **Note what you're working on** — add a comment like "Starting this today"

### While You Work

- If you finish something, **mark it complete** (check the checkbox)
- If you have a question, **add a comment** on the task
- If you find something new that needs doing, **create a new task**

### When You're Done for the Day

- **Update any in-progress tasks** with a comment about where you left off
- **Mark completed tasks** as done
- **Check if anything new** was assigned to you

## Troubleshooting

| Problem | What to Do |
|---------|-----------|
| **Can't log in** | Double-check your username and password. Try refreshing the page. If still stuck, ask Chase to reset your password. |
| **Page won't load at all** | Make sure you're on the home network. Try http://10.0.0.100:3456 directly. If PHEDRIS is down, Chase will know. |
| **I don't see any tasks** | You might not be assigned to any yet. Check that you're looking at the right project (Healing Hearts). Ask Chase if tasks need to be assigned to you. |
| **I can't find my project** | Look for "Healing Hearts" in the sidebar or project list. If it's not there, ask Chase to add you to the project. |
| **I accidentally marked something complete** | Click the checkbox again to uncheck it — tasks can be toggled back to incomplete. |
| **The task description is confusing** | Add a comment asking for clarification. Chase or the person who created the task will explain. |
| **I want to change a task's due date** | Open the task, find the due date field, and change it. Add a comment explaining why. |
| **Something else is wrong** | Ask Chase! It's better to ask than to guess. |

---

> **Remember:** Vikunja is a communication tool as much as a task tracker. If you're stuck, confused, or need help, adding a comment on the task is the fastest way to get answers.
