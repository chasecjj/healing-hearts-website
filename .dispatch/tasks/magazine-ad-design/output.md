# Photo Direction (for all ads)

**Primary photo brief — "Connection, not perfection":**
- Golden hour or warm indoor lighting
- Couples in their 30s-50s (target demographic range)
- Relaxed, genuine body language — not stiff posed portraits
- Suggest: hands touching, sitting close, walking together, foreheads near each other
- Clothing: casual but put-together (earth tones, warm colors)
- Settings: porch, living room couch, walking path, kitchen table
- Avoid: clinical settings, therapy couches, workout gear, anything that signals "medical" or "treatment"
- Avoid: overly styled editorial shoots — this should feel like a real moment
- Skin tones: diverse range across the three ads

**Secondary photo option — Jeff and Trisha:**
- If real photos of Jeff and Trisha are available, use them in Concept 2 (credibility ad)
- Should be professional but warm — not a headshot, more of an "at home" or "in conversation" shot
- Can be used at smaller size with an organic crop/mask

# Canva Execution Summary for Desirae

## What you're building:
9 total ad layouts (3 concepts x 3 sizes each). Each ad drives to the same place: a QR code linking to healingheartscourse.com/spark-challenge.

## Sizes:
1. **Quarter page:** 3.875" x 5.125" (small, text-heavy, QR code prominent)
2. **Half page horizontal:** 7.875" x 5.125" (wide, split layout with photo)
3. **Half page vertical:** 3.875" x 10.5" (tall, editorial column feel)

Set up all layouts at **300 DPI** with **0.125" bleed** on all sides. Canva's print settings should handle this — just make sure "Show bleed" is on.

## Fonts:
- **Headlines:** Cormorant Garamond, italic. If not available in Canva, use "Cormorant" or the closest serif italic. This is the signature Healing Hearts headline font.
- **Body text:** Plus Jakarta Sans, regular weight. If not available, use "Jakarta Sans" or "DM Sans" as fallback.
- **Small caps / labels:** Plus Jakarta Sans, bold, tracking widened (letter spacing +5-10%)

## Colors:
| Name | Hex | Use for |
|------|-----|---------|
| Teal | #1191B1 | Logo, credibility text, quote text, accents |
| Salmon | #B96A5F | CTA buttons/backgrounds, subheads in Concept 2, QR code surround |
| Cream | #FAF4EA | Page backgrounds |
| Charcoal | #171717 | All body text and headlines (NEVER use pure black #000000) |
| Pale teal wash | #1191B1 at 8% opacity | Quote strip backgrounds |

## QR Code:
- Generate a QR code pointing to: **healingheartscourse.com/spark-challenge**
- Minimum 1.25" x 1.25" in the layout
- Place it inside a salmon (#B96A5F) rounded rectangle with ~0.15" padding
- QR code itself should be charcoal (#171717) on white — NOT inverted, for best scan reliability
- Test scan every QR code after export

## Photo sourcing:
- Use Canva's stock photo library or Unsplash
- Search: "couple golden hour", "couple holding hands warm", "couple relaxed porch", "couple intimate moment natural"
- Apply an organic/teardrop crop mask where noted in the layouts
- If real Jeff + Trisha photos are available, use those for Concept 2

## Export:
- Export as **PDF Print** (not PNG) for magazine submission
- Verify bleed marks are included
- Provide both flattened PDF and editable Canva link to Chase for review
